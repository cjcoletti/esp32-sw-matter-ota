# Security Policy

## Supported versions

| Version | Supported |
| ------- | --------- |
| 1.0.x   | Yes       |

Older versions are not maintained.

## Scope

This project is **lab / hobby firmware** for ESP32-C6 with Matter over Wi-Fi. It is **not** a certified commercial product. Reports about missing CSA certification, test VID/PID usage, or expected lab-only behavior are out of scope for security advisories.

## Reporting a vulnerability

If you believe you have found a **security vulnerability** in this repository (for example: remote code execution, credential leakage, or unsafe default behavior that exposes devices on a network), please report it **privately**.

**Do not** open a public GitHub Issue for security reports.

### Preferred: GitHub Security Advisories

Use [Report a vulnerability](https://github.com/cjcoletti/esp32-sw-matter-ota/security/advisories/new) on this repository. GitHub allows private disclosure and coordinated discussion before any public fix.

### What to include

- Description of the issue and potential impact
- Steps to reproduce, if applicable
- Affected version (e.g. `v1.0.0`)
- Your environment (board, ESP-IDF / esp-matter versions, Matter hub) when relevant

## Response expectations

This is a **personal open-source project**. There is no official SLA, but I aim to acknowledge reports within a reasonable time and to patch confirmed issues when feasible.

Thank you for helping keep the project safer for the community.
