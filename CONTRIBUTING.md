# Contribuindo
[![Português](https://img.shields.io/badge/Language-Português-blue)](CONTRIBUTING.md)[![English](https://img.shields.io/badge/_English-inactive)](CONTRIBUTING.en.md)

Obrigado pelo interesse no **Magic Button**!

Este projeto é mantido de forma pessoal, como contribuição à comunidade Matter/ESP32. Sugestões e ideias são muito bem-vindas — elas ajudam a pensar em melhorias e casos de uso que eu talvez não tenha considerado.

## Como participar

### Sugestões e ideias

Abra uma [Issue](https://github.com/cjcoletti/esp32-sw-matter-ota/issues/new/choose) usando o template **Ideia ou sugestão**. Descreva:

- o contexto ou problema que você identificou;
- a proposta em texto (como imagina que funcionaria);
- hardware e ambiente, se relevante (placa, hub Matter, versão do ESP-IDF).

Não é necessário enviar código — prefiro entender a ideia e implementar no meu ritmo, se fizer sentido para o projeto.

### Reportar bugs

Se algo não funcionou como esperado, use o template **Reportar bug**. Inclua passos para reproduzir, ambiente e logs, se possível.

### Discussions

Você também pode usar [esse espaço](https://github.com/cjcoletti/esp32-sw-matter-ota/discussions) para trocar ideias ou tirar dúvidas de forma mais aberta.

## O que não aceito

**Pull Requests** — por favor, não envie PRs com alterações de código ou firmware. Prefiro implementar mudanças eu mesmo, no meu ritmo.

Exceção: se encontrar um erro claro de digitação na documentação (`README`, `docs/`), pode abrir uma Issue apontando o trecho; avalio caso a caso.

## Forks

Você pode fazer **fork** livremente e adaptar o projeto às suas necessidades, conforme a [licença MIT](LICENSE). Se publicar uma variante interessante, fique à vontade para mencionar na Issue ou Discussion — fico feliz em saber como outros estão usando o projeto.

## Expectativas

- Não há suporte oficial, SLA de resposta nem garantia de implementação.
- Nem toda sugestão será adotada — e tudo bem. Agradeço o tempo de quem compartilha ideias.
- Respondo quando possível, no tempo que o projeto pessoal permite.
